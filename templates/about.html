Technologies We Use

Python & Flask

Backend powered by Python with Flask framework

Implementation Details

We built a RESTful API using Flask to handle prediction requests. The backend processes input data and returns predictions from our trained ML models.

Machine Learning

Advanced regression models for predictions

Implementation Details

We used Scikit-learn to train regression models on medical insurance data. Features include age, BMI, smoking status, and region to predict costs.

Tailwind CSS

Modern CSS framework for responsive design

Implementation Details

We used Tailwind's utility classes to quickly build a responsive interface. Custom animations and transitions were added for better user experience.
